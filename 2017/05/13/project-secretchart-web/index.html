시크릿차트 웹 버전 스크린샷

개요

주식회사 이큐브랩 재직 중 시크릿차트 팀으로 배정받은 이후로 처음으로 맡은 프로젝트 입니다. 기존의 시크릿차트 서비스의 기획을 부분 차용하여 새로운 서비스 기획 이후에 서비스 개발도 처음부터 다시 하게 되었습니다. 성형 큐레이션 및 소셜 커뮤니티 서비스 이며 https://www.secretchart.com 에 배포되어 서비스되고 있습니다.

개발 기간

2016.08 ~ 2016.12

기여도

100%

사용 기술

시크릿차트 웹 애플리케이션을 개발에 사용한 기술 목록입니다.

  • Laravel : 웹 풀스택 프레임워크
  • Lumen : REST API를 위한 마이크로 프레임워크
  • jQuery : DOM 조작 및 동적 웹페이지를 위해 사용한 라이브러리
  • Semantic-UI : CSS 프레임워크
